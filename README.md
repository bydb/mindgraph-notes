# MindGraph Notes

A modern note-taking application with graph visualization, built with Electron and React.

## Features

- Markdown editor with live preview
- Wiki-style linking between notes (`[[note]]`)
- Interactive graph visualization of note connections
- PDF viewer integration
- Zotero integration for academic references
- AI-powered features (context menu, image generation)
- Template system for quick note creation
- Full-text search across all notes
- Backlinks panel showing incoming references

## Tech Stack

- **Electron** - Cross-platform desktop app
- **React** - UI framework
- **TypeScript** - Type-safe development
- **CodeMirror** - Markdown editor
- **React Flow** - Graph visualization
- **Zustand** - State management

## Development

```bash
cd app
npm install
npm run dev
```

## Build

```bash
cd app
npm run build
```
