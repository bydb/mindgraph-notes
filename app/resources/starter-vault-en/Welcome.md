# Welcome to MindGraph Notes!

This is your **Starter Vault** with example notes that showcase the most important features.

## Getting Started

1. Check out the examples in the [[Canvas Example|Examples]] folder
2. Create your first note with `Cmd+K`
3. Link notes with `[[Wikilinks]]`
4. Discover the Canvas in Split mode

## Folder Structure

- **Examples/** - Example notes for various features
- **Templates/** - Space for your own templates

## Useful Shortcuts

| Shortcut | Function |
|----------|----------|
| `Cmd+K` | Quick Switcher |
| `Cmd+P` | Quick Search |
| `Cmd+,` | Settings |
| `Cmd+Shift+T` | Insert Template |

Have fun exploring!
