# Getting Started

Welcome to MindGraph Notes! Here you'll learn the basics.

## The Editor

You're looking at the **Editor** right now. You write in Markdown on the left, and the formatted preview appears on the right.

Here's how Markdown works:
- `**bold**` becomes **bold**, `*italic*` becomes *italic*
- Headings with `#`, `##`, `###`
- Lists with `-` or `1.`
- Tables, LaTeX formulas, and diagrams are supported

> **Try it:** Click "Editor" and start typing. Also check out the [[Markdown Showcase]] for all possibilities.

## Essential Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+N` | Create new note |
| `Cmd+K` | Quick search — open a note |
| `Cmd+Shift+T` | Template Picker |
| `Cmd+E` | Cycle editor mode |
| `Cmd+,` | Settings |
