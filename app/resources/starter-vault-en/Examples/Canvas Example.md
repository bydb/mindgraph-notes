# Canvas Example

The **Canvas** displays your notes as an interactive graph. Each note becomes a node, each [[Welcome|Wikilink]] becomes a connection.

## How to Use the Canvas

1. Switch to **Split mode** (titlebar)
2. All linked notes appear as connected cards
3. Click a card to open the note
4. Drag cards to visually organize your knowledge

## Features

- **Drag & Drop** - Position cards freely
- **Colors** - Right-click a card > Change color
- **Layouts** - Automatic arrangement (Hierarchical, Grid)
- **Export** - Export canvas as SVG
