# Project Planning

This note shows how to manage tasks and deadlines in MindGraph. See also [[01 - Getting Started]] for the basics.

## Tasks with Due Dates

MindGraph recognizes tasks with dates automatically and displays them in the **Overdue Tasks** panel.

- [ ] Finish research @[[2026-03-25]]
- [ ] Write draft @[[2026-03-28]]
- [ ] Get feedback @[[2026-04-01]]
- [x] Choose topic @[[2026-03-20]]

> **Tip:** The clock icon in the top right shows all due tasks at a glance.

## Linking Project Notes

A project often consists of multiple notes. Link them:

- Research findings — create a new note
- Ideas and brainstorming — [[Knowledge Hub]]
- Presentation material — [[Markdown Showcase]]

## Using Templates

For recurring projects you can create **templates**:
1. Press `Cmd+Shift+T`
2. Choose a fitting template
3. Customize it for your project
