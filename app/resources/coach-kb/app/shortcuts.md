# Tastenkürzel (macOS: ⌘ / Windows + Linux: Strg)

## Allgemein

- **⌘ + N** — Neue Notiz im aktuellen Ordner
- **⌘ + P** — Schnellsuche (Notizen, Tags, Wikilinks)
- **⌘ + ,** — Einstellungen öffnen
- **⌘ + /** — Slash-Command-Menü erzwingen
- **⌘ + S** — Notiz speichern (passiert auto, aber für Sofort-Save)
- **⌘ + ⇧ + N** — **Schnellerfassung** (separates Mini-Fenster zum
  schnellen Reinwerfen einer Idee — Default-Shortcut, in Settings änderbar)

## Editor

- **⌘ + B** / **⌘ + I** — Fett / Kursiv
- **⌘ + K** — Link einfügen
- **⌘ + ⇧ + Z** — Zotero-Picker (wenn Zotero-Modul aktiv)
- **⌘ + D** in der Schnellerfassung — Diktat (Whisper STT) starten

## Navigation

- **⌘ + 1 / 2 / 3** — Editor / Split / Canvas-View
- **⌘ + B** im Sidebar-Kontext — Sidebar ein/ausblenden
- **⌘ + ⇧ + F** — Volltextsuche im Vault

## Wechsel zwischen Editor-Modi (pro Notiz)

Oben rechts in der Notiz: drei Knöpfe für **Markdown / Schreiben / Lesen**.
Default kannst du in Settings → Editor → "Default-Ansicht" festlegen.

## Tab-Verwaltung

- **⌘ + W** — aktuellen Tab schließen
- **⌘ + ⇧ + T** — geschlossenen Tab wiederherstellen
- **⌘ + ⌥ + Pfeil** — zwischen Tabs wechseln

Hinweis: Manche Shortcuts sind plattformspezifisch oder vom System
belegt — schau in Settings → Allgemein nach.
