# LanguageTool

## Wann nutzen
Beim Schreiben längerer Texte (Artikel, Berichte, Konzepte), um
Grammatik- und Rechtschreibfehler direkt im Editor unterringeln zu
lassen — inkl. Verbesserungsvorschlägen.

## Wie aktivieren
Einstellungen → Module → "LanguageTool". Wahlweise lokal (LanguageTool-
Server selbst hosten) oder über die LanguageTool-Cloud-API mit Key.

## Beispiel
Während du schreibst, werden Fehler rot/blau unterringelt. Klick auf
einen Vorschlag übernimmt die Korrektur. Sprachwechsel in den Einstellungen.
