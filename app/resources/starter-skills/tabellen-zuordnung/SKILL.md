# Daten aus Tabellen zuordnen

## Wann anwenden

Wenn Spalten aus einer Tabelle extrahiert, zwei Tabellen über ein gemeinsames Merkmal
verknüpft oder Daten in eine neue Struktur gebracht werden sollen.

## Vorgehen

1. Lies ALLE beteiligten Tabellen vollständig, bevor du zuordnest.
2. Bestimme das Verknüpfungsmerkmal (z. B. ein Name oder eine Nummer, die in beiden
   Tabellen vorkommt). Prüfe auf Schreibvarianten (Groß-/Kleinschreibung, Leerzeichen).
3. Baue die Zieltabelle: erste Zeile sind sprechende Spaltenüberschriften,
   danach eine Zeile pro Datensatz.
4. Zellen ohne Treffer bleiben nicht leer, sondern erhalten eine eindeutige
   Markierung (z. B. „nicht gefunden") — so sind Lücken sofort sichtbar.
5. Nenne in deiner Abschlussantwort die Anzahl der zugeordneten und der
   nicht zugeordneten Zeilen.

## Regeln

- Niemals Werte erfinden oder raten — was in den Quellen nicht steht, wird als
  fehlend markiert.
- Reihenfolge der Quelltabelle beibehalten, sofern der Auftrag nichts anderes sagt.
- Keine zusammengeführten Zellen, keine Formatierungs-Spielereien: eine saubere,
  filterbare Tabelle.
- Bei Unklarheiten im Verknüpfungsmerkmal die konservative Wahl treffen und die
  Annahme in der Abschlussantwort benennen.
