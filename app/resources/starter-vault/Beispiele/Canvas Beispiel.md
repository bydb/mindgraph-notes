# Canvas Beispiel

Der **Canvas** zeigt deine Notizen als interaktiven Graphen. Jede Notiz wird zu einem Knoten, jeder [[Willkommen|Wikilink]] zu einer Verbindung.

## So nutzt du den Canvas

1. Wechsle in den **Split-Modus** (Titlebar)
2. Alle verlinkten Notizen erscheinen als verbundene Karten
3. Klicke auf eine Karte um die Notiz zu öffnen
4. Ziehe Karten um dein Wissen visuell zu ordnen

## Features

- **Drag & Drop** - Karten frei positionieren
- **Farben** - Rechtsklick auf eine Karte > Farbe ändern
- **Layouts** - Automatische Anordnung (Hierarchisch, Raster)
- **Export** - Canvas als SVG exportieren
