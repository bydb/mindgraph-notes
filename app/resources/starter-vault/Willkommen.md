# Willkommen bei MindGraph Notes!

Dies ist dein **Starter-Vault** mit Beispielnotizen, die dir die wichtigsten Features zeigen.

## Erste Schritte

1. Schau dir die Beispiele im Ordner [[Canvas Beispiel|Beispiele]] an
2. Erstelle deine erste eigene Notiz mit `Cmd+K`
3. Verlinke Notizen mit `[[Wikilinks]]`
4. Entdecke den Canvas im Split-Modus

## Ordnerstruktur

- **Beispiele/** - Beispielnotizen zu verschiedenen Features
- **Templates/** - Platz für deine eigenen Vorlagen

## Nützliche Tastenkürzel

| Kürzel | Funktion |
|--------|----------|
| `Cmd+K` | Quick Switcher |
| `Cmd+P` | Schnellsuche |
| `Cmd+,` | Einstellungen |
| `Cmd+Shift+T` | Template einfügen |

Viel Spaß beim Entdecken!
