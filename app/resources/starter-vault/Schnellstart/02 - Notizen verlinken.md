# Notizen verlinken

Das Besondere an MindGraph: Du kannst Notizen miteinander **verlinken** — wie Wikipedia, nur fuer dein eigenes Wissen.

## So funktioniert's

1. Tippe `[[` im Editor
2. Waehle eine Notiz aus der Liste
3. Fertig! Der Link sieht so aus: [[Willkommen]]

## Warum verlinken?

- Dein Wissen wird vernetzt statt in Ordnern versteckt
- Du findest Zusammenhaenge zwischen Themen
- **Backlinks** zeigen dir automatisch, welche Notizen hierher verlinken

> **Tipp:** Verlinke verwandte Themen beim Schreiben sofort — so entsteht dein persoenliches [[Wissensnetz]].
