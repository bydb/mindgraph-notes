# Atomare Notizen

Eine atomare Notiz enthält genau **einen** Gedanken — so kurz wie möglich, so lang wie nötig.

## Warum klein besser ist

- **Verlinkbar**: Auf einen präzisen Gedanken kann man verweisen, auf ein 20-Seiten-Dokument nicht sinnvoll
- **Wiederverwendbar**: Derselbe Baustein passt in verschiedene Kontexte (Vortrag, Artikel, Fortbildung)
- **Wartbar**: Eine kleine Notiz lässt sich korrigieren, ohne ein Gesamtdokument umzubauen

## Faustregel

Wenn du einer Notiz keinen präzisen Titel geben kannst, enthält sie mehr als einen Gedanken — teile sie auf.
