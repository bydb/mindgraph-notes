Was MindGraph Notes besonders macht

Karteikarten & Spaced Repetition

SM-2 Algorithmus, KI-Quiz-Generierung aus Notizen, Anki-Import, Statistik-Dashboard mit Streak und Heatmap. Kein extra Tool noetig.

Kostenloser E2E Sync

AES-256-GCM Verschluesselung, Zero-Knowledge Server. Kein Account noetig, nur Vault-ID + Passphrase. Obsidian Sync kostet $8/Monat.

Integriertes KI-Terminal

OpenCode, Claude Code oder Ollama direkt neben deinen Notizen. KI liest deinen Vault und arbeitet im Kontext.

E-Mail-Inbox mit KI

IMAP-Abruf, automatische Relevanzanalyse via Ollama, relevante E-Mails werden als Markdown-Notizen gespeichert.

Obsidian-kompatibel

Gleiche Markdown-Dateien, Wikilinks, Callouts, Frontmatter. Du kannst deinen Obsidian-Vault direkt oeffnen.

100% Open Source

AGPL-3.0 Lizenz. Kein Vendor Lock-in, keine Telemetrie, keine versteckten Kosten. Dein Wissen gehoert dir.
