# Unterrichtsentwurf erstellen

## Wann anwenden

Wenn eine Unterrichtsstunde, ein Workshop oder eine Fortbildungseinheit geplant
werden soll — aus einem Thema, Material oder Stichpunkten.

## Vorgehen

1. Kläre aus der Quelle: Thema, Zielgruppe/Stufe, verfügbare Zeit, vorhandenes
   Material und Vorwissen. Fehlendes als Annahme kennzeichnen.
2. Formuliere zwei bis vier Lernziele nach dem Muster „Die Teilnehmenden können …"
   — beobachtbar und überprüfbar, keine vagen Absichtserklärungen.
3. Plane den Verlauf als Tabelle mit den Spalten Phase / Zeit / Was passiert /
   Sozialform / Material. Übliche Phasen: Einstieg, Erarbeitung, Sicherung,
   optional Vertiefung oder Transfer.
4. Die Zeiten müssen sich auf die verfügbare Gesamtzeit summieren; plane einen
   Puffer ein und kennzeichne, welche Phase bei Zeitnot gekürzt wird.
5. Abschnitt „Material": alles, was vorbereitet werden muss, als Checkliste.
6. Abschnitt „Differenzierung": je ein Angebot für Schnelle und für Unterstützung.

## Regeln

- Aktivierung vor Vortrag: Phasen so planen, dass die Teilnehmenden arbeiten,
  nicht nur zuhören.
- Jede Phase zahlt erkennbar auf ein Lernziel ein — Phasen ohne Ziel streichen.
- Realistische Zeiten: Einstieg und Sicherung werden regelmäßig unterschätzt.
- Keine Methodenvielfalt um ihrer selbst willen.
