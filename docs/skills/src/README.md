# Skill-Katalog — Quellen

Die `*.md`-Dateien hier sind die Quellen des kuratierten Skill-Katalogs
(SKILL.md-Format, agentskills.io). Die App lädt NICHT diese Dateien, sondern
`../index.json` — bewusst JSON, damit GitHub Pages/Jekyll die Inhalte roh
ausliefert statt Markdown zu HTML zu verarbeiten.

Nach Änderungen an den Quellen `index.json` neu generieren (Skript siehe
`docs/agent-skills-plan.md` bzw. das Python-Snippet im Commit, der diese
Datei eingeführt hat) und `updated` hochzählen.
