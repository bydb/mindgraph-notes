KI direkt in deinen Notizen

Das integrierte Terminal macht MindGraph Notes einzigartig: Starte OpenCode, Claude Code oder Ollama direkt neben deinen Notizen. Lass dir Grafiken erstellen, Zusammenfassungen schreiben oder Quiz-Fragen generieren – ohne Kontextwechsel.

  • ✓ OpenCode, Claude Code und Ollama direkt nutzen
  • ✓ KI liest deine Notizen und arbeitet im Kontext
  • ✓ Grafiken, Zusammenfassungen, Quiz – alles per Prompt
  • ✓ 100% lokal möglich – deine Daten bleiben privat

Dataview Queries

Frage deine Notizen ab wie eine Datenbank

Mit Dataview kannst du Notizen nach Metadaten abfragen und als Listen oder Tabellen darstellen. Schreibe Queries im Edit-Modus und wechsle zu Live-Preview, um die Ergebnisse zu sehen.

Grundsyntax

```dataview
LIST
```
Listet alle Notizen auf 
```dataview
TABLE status, priority, deadline
```
Tabelle mit Frontmatter-Spalten

FROM - Quellen filtern

LIST FROM #projekt
Nach Tag filtern 
LIST FROM "Work/Projects"
Nach Ordner filtern 
LIST FROM #projekt AND "Work"
Quellen kombinieren

WHERE - Bedingungen

LIST WHERE status = "active"
Text vergleichen 
LIST WHERE priority >= 2
Zahlen vergleichen 
LIST WHERE completed = false
Boolean prüfen 
LIST WHERE contains(tags, "urgent")
In Arrays suchen

SORT & LIMIT

LIST SORT file.mtime DESC
Nach Datum sortieren 
LIST SORT priority ASC LIMIT 10
Sortieren und begrenzen

Verfügbare Felder

Datei-Felder

  • file.name - Dateiname ohne Endung
  • file.path - Relativer Pfad
  • file.folder - Übergeordneter Ordner
  • file.ctime - Erstelldatum
  • file.mtime - Änderungsdatum
  • file.tags - Alle Tags

Frontmatter-Felder

Alle Felder aus dem YAML-Header sind direkt verfügbar:

  • status, priority, deadline
  • author, tags, category
  • ...und alle eigenen Felder

Vollständiges Beispiel

```dataview
TABLE status, deadline, priority
FROM "Work/Projects"
WHERE !completed AND priority >= 2
SORT deadline ASC
LIMIT 10
```

Zeigt eine Tabelle mit Status, Deadline und Priorität für alle nicht abgeschlossenen Projekte mit Priorität ≥ 2, sortiert nach Deadline.

Properties Panel

YAML-Frontmatter komfortabel bearbeiten

Das Properties Panel zeigt alle YAML-Frontmatter-Felder oberhalb des Editors an. Bearbeite Metadaten ohne YAML-Syntax – mit passenden Eingabefeldern für jeden Datentyp.

Booleans

Checkboxen für true/false Werte wie completed oder published

Datum

Datumspicker für Felder im ISO-Format (YYYY-MM-DD)

Zahlen

Zahlenfelder für priority, rating etc.

Text & Arrays

Textfelder für Strings, kommagetrennte Listen für Arrays

Tag-Autocomplete

Beim Tags-Feld werden alle existierenden Vault-Tags als Vorschläge angezeigt

Beispiel Frontmatter

---
title: Mein Projekt
status: active
priority: 2
deadline: 2026-03-15
completed: false
tags: [projekt, arbeit, wichtig]
---

Diese Felder werden im Properties Panel als Textfeld, Dropdown, Zahl, Datumspicker, Checkbox und Array-Input angezeigt.

Tools einrichten

So holst du das Maximum aus MindGraph Notes heraus

Claude Code

Anthropics offizieller KI-Coding-Agent. Claude Code ist der mächtigste KI-Assistent für Entwickler.

  1. Node.js installieren: brew install node
  2. Claude Code installieren: npm i -g @anthropic-ai/claude-code
  3. API-Key von console.anthropic.com holen
  4. Key setzen: export ANTHROPIC_API_KEY="sk-ant-..."
  5. Starten: claude

Tipp: Key permanent speichern mit echo 'export ANTHROPIC_API_KEY="..."' >> ~/.zshrc

Mehr erfahren →

OpenCode

Open-Source KI-Coding-Agent. Unterstützt verschiedene LLM-Anbieter wie OpenAI, Anthropic, Gemini und lokale Modelle.

  1. OpenCode installieren: curl -fsSL https://opencode.ai/install | bash
  2. API-Key setzen (z.B. OpenAI): export OPENAI_API_KEY="sk-..."
  3. Starten: opencode

Unterstützt auch lokale Modelle via Ollama - keine API-Kosten!

Mehr erfahren →

Ollama

Lokale LLMs wie Llama, Mistral oder Gemma auf deinem Mac. Keine Cloud, keine API-Kosten, volle Privatsphäre.

Installation: brew install ollama

Nach der Installation: ollama pull llama3 um ein Modell zu laden.

Mehr erfahren →

Zotero

Literaturverwaltung für Wissenschaft und Recherche. Füge Zitate direkt in deine Notizen ein.

  1. Zotero installieren von zotero.org
  2. Better BibTeX Plugin installieren für Export
  3. In MindGraph Notes: Einstellungen → Zotero Pfad setzen
Mehr erfahren →

Flux2 Bildgenerierung

Generiere Bilder direkt im Terminal mit Flux2. Lokale KI-Bildgenerierung für deine Notizen - läuft über Ollama.

Verwendung: ollama run x/flux2-klein

Voraussetzung: Ollama muss installiert sein. Das Modell wird beim ersten Start automatisch heruntergeladen.

Mehr erfahren →

Docling PDF-Extraktion

Extrahiere automatisch Text, Tabellen und Bilder aus PDFs. Konvertiert zu sauberem Markdown – lokal auf deinem Rechner.

Installation: docker run -p 5001:5001 ds4sd/docling-serve

Voraussetzung: Docker muss installiert sein. Docling läuft dann auf localhost:5001. Aktiviere OCR in den Einstellungen für gescannte PDFs.

Mehr erfahren →

LanguageTool Grammatikprüfung

Prüfe deine Texte auf Grammatik, Rechtschreibung und Stil. Lokal via Docker oder mit LanguageTool Premium für erweiterte Funktionen.

Installation: docker run -d -p 8010:8010 erikvl87/languagetool

Docker-Version für lokale Nutzung. Alternativ: LanguageTool Premium API mit Username + API-Key in den Einstellungen konfigurieren.
